# Crush CLI - by Charm

## Installation

```shell
# Install with npm
npm install -g @charmland/crush

# update to latest version with npm
npm update -g @charmland/crush
```

> via [github.com/charmbracelet/crush](https://github.com/charmbracelet/crush/)


## Get Version

```
% crush --version
```

## Usage

Crush is a CLI coding assistant from Charm, known for creating beautiful terminal applications. It provides an AI-powered coding experience directly in your terminal.

```
Usage: crush [options] [prompt]

Options:
  -v, --version      Display version information
  -h, --help         Display help for command
```
